DT228 Year1 Modules

Semester 1

The semester 1 of computer science course contains the following modules.

  • Communication
  • Web developement
  • Programming
  • Information tech fundamentals
  • Mathematics 1
  • Program design

Semester 2

The semester 2 of computer science course contains the following modules.

  • Algorithm design and problems
  • Programming
  • Information tech fundamentals
  • Computer arcitecture and technology
  • Data exploration
  • Microprocessor systems
  • Operating systems 1
  • Program with presistent data
Each semester has an exam at the end which tests the student of the material learned in that semester.
